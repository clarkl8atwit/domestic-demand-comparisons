# ✈️ Domestic Flight Visualization

### [Click here to view website](https://clarkl8atwit.github.io/domestic-demand-comparisons/)

---


### Overview
This project visualizes **domestic flight demand** across major U.S. cities.

- Lets users see a city's **outbound flights**, weighted by **how many people use them per year**.
- Allows users to **select a specific year** to compare how demand changes over time.

---

### How to Use

1. **Select a city’s circle** on the map. 
2. **Use the dropdown menu** on the right to view data for a specific year.  
3. To change cities, **click the same city again** to return to the loading page, or **click another city** to view its routes.

---
